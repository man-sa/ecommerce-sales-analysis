# ecommerce-sales-analysis
E-commerce sales analysis using SQL and Power BI dashboard.

## Overview

This project analyzes an e-commerce dataset to extract meaningful business insights using SQL and Power BI.

## Tools & Technologies

* SQL (PostgreSQL)
* Power BI
* Excel / CSV Dataset

## Key KPIs

* Total Revenue
* Total Orders
* Total Customers
* Average Order Value (AOV)

## Dashboard Features

* Monthly Sales Trend
* Top Products Analysis
* Payment Method Distribution
* Customer Insights

## SQL Analysis

Performed data analysis using SQL queries such as:

* Revenue calculation
* Order count
* Customer analysis
* Average order value
* Monthly trends
